====== Init SD card ======
Created Wednesday 16 April 2014

**-Init SD Card**

sudo ./mkcard.sh /dev/sdb

**-Init Boot Sector and Kernel**

sudo cp x-loader/MLO /media/boot/
sync
sudo cp u-boot-release/u-boot.bin /media/boot/
sync
sudo cp uImage /media/boot/
sync

**-Init Filesystem**
